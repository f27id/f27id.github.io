Steps: * 1. Create a new prototype (web based) with two Screens: Screen 1 and Screen 2 * 2. Include an input text field and button in Screen 1 * 3. Include a label in Screen 2 with message: “Successful” Main Aim: Screen 1 will transition to Screen 2 only when the text “abcd” is present in the text box * 4.
